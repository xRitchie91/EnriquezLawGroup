# Enriquez Law Group - Law Firm Website

A modern, responsive website for a professional law firm built with React, Vite, and Tailwind CSS.


## 🌟 Features

- **Responsive Design** - Fully optimized for mobile, tablet, and desktop devices
- **Modern UI/UX** - Clean, minimalist design with professional styling
- **Smooth Navigation** - Seamless scrolling between sections
- **Interactive Elements** - Hover effects, animations, and transitions
- **Contact Form** - Functional form for client inquiries
- **Practice Areas** - Showcase of legal services offered
- **Mobile Menu** - Hamburger menu for mobile devices
- **Professional Icons** - Lucide React icons for consistent design

## 📋 Sections

1. **Navigation Bar** - Fixed header with smooth scrolling links
2. **Hero Section** - Eye-catching headline with call-to-action buttons
3. **Practice Areas** - Six legal service categories with descriptions
4. **About Section** - Firm history, statistics, and core values
5. **Contact Section** - Contact form and business information
6. **Footer** - Links, social media, and legal disclaimer

## 🛠️ Technologies Used

- **React** - JavaScript library for building user interfaces
- **Vite** - Next-generation frontend build tool
- **Tailwind CSS** - Utility-first CSS framework
- **Lucide React** - Beautiful & consistent icon library
- **JavaScript (ES6+)** - Modern JavaScript features


## 📱 Responsive Breakpoints

- **Mobile**: < 768px
- **Tablet**: 768px - 1024px
- **Desktop**: > 1024px
